152. Maximum Product Subarray
152. 最大乘積子陣列


Given an integer array nums, find a
給定一個整數陣列 nums，找到一個
subarray 子陣列
 that has the largest product, and return the product.
擁有最大的產品，並退回產品。

The test cases are generated so that the answer will fit in a 32-bit integer.
生成測試用例，以便答案適合 32 位整數。
--------------------------------------------
public class Solution {
    public int maxProduct(int[] nums) {
        if (nums == null || nums.length == 0) {
            return 0; // Edge case: no elements
        }

        // Initialize the variables
        int maxProduct = nums[0];
        int minProduct = nums[0];
        int globalMax = nums[0];

        for (int i = 1; i < nums.length; i++) {
            int num = nums[i];

            // If num is negative, swap maxProduct and minProduct
            if (num < 0) {
                int temp = maxProduct;
                maxProduct = minProduct;
                minProduct = temp;
            }

            // Update maxProduct and minProduct
            maxProduct = Math.max(num, num * maxProduct);
            minProduct = Math.min(num, num * minProduct);

            // Update globalMax
            globalMax = Math.max(globalMax, maxProduct);
        }

        return globalMax;
    }
}
